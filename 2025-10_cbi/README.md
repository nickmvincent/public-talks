# CBI Talk - Collective Bargaining in the Information Economy

NeurIPS Position Papers 2025

## Files

- **`index.html`** - Minimalist website for the talk
- **`poster.tex`** - LaTeX poster with the same content
- **`main.md`** - Marp slides source

## Usage

### View Website

```bash
open index.html
```

### Generate PDF Poster

```bash
pdflatex poster.tex
```
